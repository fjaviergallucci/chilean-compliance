# Ley 21.719 — Amendments Changelog (reference only)

**Published:** 2024-12-13
**In force:** 2026-12-01
**Source PDF:** `sources/Ley-21719_13-DIC-2024.pdf`

Ley 21.719 is largely a meta-law amending Ley 19.628. **The operative law
after 2026-12-01 lives in `corpus/19628-data-protection-consolidated/`.**
This directory is reference-only: it lists each amendment in source order
and points to the article in the consolidated 19.628 that it produced.

## Files
- `_amendments.json` — machine-parsed amendment items (regenerated by `tools/parse_21719_amendments.py`).
- `changes.md` — human-readable changelog with links into the consolidated corpus (produced in F2).

## Do not cite this as operative law.
When citing data-protection obligations, cite the **consolidated** 19.628
article (with footer noting the amending 21.719 provision and effective date).
